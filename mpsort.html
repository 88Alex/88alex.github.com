The Mean-Partition Sort Algorithm

The Mean-Partition Sort Algorithm is a convenient sorting algorithm. It is a variation on the standart partition-sort algorithm, except instead of picking the first item as the partition, it selects the mean of all the items.

A pseudo-code implementation would be:

sort(array): if length(array) < 2: do nothing if length(array) == 2: if array[0] > array[1]: swap(array[0], array[1]) if isSorted(array) do nothing for element in array: if element < mean(array): list1.append(element) else if element == mean(array): list2.append(element) else list3.append(element) sort(list1) sort(list3) array = list1 + list2 + list3
